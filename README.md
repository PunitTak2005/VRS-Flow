# 🚀 Volunteer Management System

A full-stack web application designed to efficiently manage volunteers, events, attendance, and communication in an organized and scalable way.

This project is fully containerized using Docker and supports production-ready deployment with CI/CD compatibility.

---

## 🧩 Tech Stack

### Frontend
- React / Vite  
- HTML, CSS, JavaScript  
- Responsive UI design  

### Backend
- Node.js  
- Express.js  
- REST APIs  
- JWT Authentication  

### Database
- MongoDB  

### DevOps
- Docker  
- Docker Compose  
- GitHub Actions (CI/CD ready)  

---

## ✨ Key Features

### 🔐 Authentication
- OTP-based login system  
- Secure JWT authentication  
- Role-based access (Admin/User)  

### 👥 User Management
- Add / update / suspend users  
- Admin control dashboard  

### 📅 Event Management
- Create and manage events  
- Assign volunteers  
- Track participation  

### 📊 Attendance System
- Mark attendance in real time  
- Generate reports  

### 📄 PDF Generation
- Export reports as PDFs  
- Certificates and attendance reports  

### 📧 Email Notifications
- Automated email system for updates and alerts
